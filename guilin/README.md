### 170815(Thurs)
- 题目：输入两个正整数m和n，求其最大公约数和最小公倍数。
- 用时：28分钟。
- 测试结果：```10、6两个数的最大公约数为2，最小公倍数为30。
          5、7两个数的最大公约数为1，最小公倍数为35。```
- 其中的问题：最开始的思路是算出公倍数数组就行，可以看出初中的数学知识竟然忘了。理清思路```每个数约数数组 => 约数数组相同部分 => 相同部分的最小值就是最小公约数 => 约束数组相同部分之积*每个数除去约束数组的值就是最大公倍数```后代码就敲得很顺畅了。
